```javascript
var a = {n: 1}
var b = a;
a.x = a = {n: 2};
console.log(a.x);
console.log(b.x)
```

由于`.`运算符优先级高于赋值运算符，因此执行顺序为：

1. 给`a`添加属性`x`，指向`undefined`。

   此时：

   ```javascript
   a;//{x:undefined,n:1}
   b;//{x:undefined,n:1}
   ```

2. 将`{n:2}`赋值给`a`。

   此时：

   ```javascript
   a;//{n:2}
   b;//{x:undefined,n:1}
   ```

3. 第二步后的`a.x`代表的是旧的`a`的`x`属性（`b`仍然指向旧的`a`），将其赋值为新的`a`。

   此时：

   ```javascript
   a;//{n:2}
   b;//{x:{n:2},n:1}
   ```
