==*==赋值运算符为从右到左运算

大致优先级表：

| 运算符                                                 | 描述                                             |
| ------------------------------------------------------ | ------------------------------------------------ |
| ` ()`                                                  | 圆括号优先级最高                                 |
| `. [] ()  new ...(...)`                                | 成员/字段访问、数组下标、函数调用、new带参数列表 |
| `new ...`                                              | new不带参数列表                                  |
| `...++   ...--`                                        | 后置递增/递减                                    |
| `++... --... ! ~ - +... -... delete await typeof void` | 前置递增/递减、一元运算符                        |
| `...**...`                                             | 幂                                               |
| `* / %`                                                | 乘法、除法、取模                                 |
| `...+... ...-...`                                      | 加法、减法                                       |
| `<< >> >>>`                                            | 按位左移、按位右移、无符号右移                   |
| `< <= > >= in instanceof`                              | 大小比较                                         |
| `== != === !==`                                        | 等号、不等号                                     |
| `...&...`                                              | 按位与                                           |
| `...^...`                                              | 按位异或                                         |
| `...|...`                                              | 按位或                                           |
| `...&&...`                                             | 逻辑与                                           |
| `...||...`                                             | 逻辑或                                           |
| `...?...:...`                                          | 三元运算符                                       |
| `= += -= *= /= %= <<= >>= >>>= &= ^= |=`               | 赋值                                             |
| `yield yield*`                                         | yield                                            |
| `...`                                                  | 展开运算符                                       |
| `,`                                                    | 逗号                                             |
