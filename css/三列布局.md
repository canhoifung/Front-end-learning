# 双飞翼布局
![双飞翼布局](images/20190313182150051_26077.png =599x)  
```html
//需要优先写放置在中间的板块,再左右
<div class="main">
            <div class="main-content">main content</div>
</div>
<div class="left">left</div>
<div class="right">right</div>
```
```css
<style>
    *{
        margin:0;
        padding: 0
    }
    .main{
        float: left;
        width: 100%;
    }
    .main .main-content{
        //利用这个来实现居中,并且能跟随main元素宽度变化而变化,在于左右margin为左右部分的宽度200px+间隔10px
        margin: 0 210px;
        background-color: rgba(33, 114, 214, 0.8);
        height: 500px
    }
    .left{
        width: 200px;
        float: left;
        background-color: rgba(255, 82, 0, 0.8);
        //百分比时为以父元素的content的长度的百分比,此处-100%指该元素的左边框距父元素的右边框的距离,left左边框与container左边框重合
        margin-left: -100%;
        height: 200px
    }
    .right{
        width: 200px;
        height: 200px;
        margin-left: -200px;  //此处指right元素的宽度
        float: left;
        background-color: rgba(90, 243, 151, 0.8);
    }
<style>
```

# 圣杯布局
```html
<style>
*{margin:0;padding:0;}
body{background-color: #cccc;text-align:center;}
header{height:50px;background-color: pink;line-height:50px;}
.container{padding:0 220px 0 200px;}
.middle,.left,.right{
    position:relative;
    float: left;
    min-height: 300px;
}
.middle{background-color: red;width:100%;}
.left{width:200px;background-color: yellow;margin-left:-100%;left:-200px;}
.right{width:220px;background-color: blue;margin-left:-220px;right:-220px;}
</style>

<header>
    <h2>我是header</h2>
</header>
<div class="container">
    <div class="middle">
        <h4>我是middle</h4>
        <p>我是内容部分我是内容部分我是内容部分我是内容部分我是内容部分我是内容部分我是内容部分我是内容部分</p>
    </div>
    <div class="left">
        <h4>我是left</h4>
        <p>我是left的内容</p>
    </div>
    <div class="right">
        <h4>我是right的内容</h4>
        <p>我是right的内容</p>
    </div>
</div>
```

# Flex布局
```css
<style type="text/css">
.container{
    display: flex;
    min-height: 130px;
}
.main{
    flex-grow: 1; //设置main的放大比例,将空余空间用main填充,默认0即不作处理
    background-color: blue;
}
.left{
    order: -1;    //order属性越小排列越靠前,默认0
    flex-basis: 200px;  //设置固定宽度
    background-color: green;
}
.right{
    flex-basis: 300px; //设置固定宽度
    background-color: red;
}
</style>
```
```html
<div class="container">
　　<div class="main">main</div>
　　<div class="left">left</div>
　　<div class="right">right</div>
</div>
```
# 绝对定位布局
```css
.container{
        position: relative;
    }
    .main,.right,.left{
        top: 0;
        height: 130px;
    }
    .main{
        margin: 0 300px 0 200px;
        background-color: blue;
    }
    .right{
        position: absolute;
        width: 300px;
        right: 0;
        background-color: red;
    }
    .left{
        position: absolute;
        width: 200px;
        background-color: green;
        left: 0;
    }
```
```html
<div class="container">
　　<div class="main">main</div>
　　<div class="left">left</div>
　　<div class="right">right</div>
</div>
```
